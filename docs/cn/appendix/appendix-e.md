## 腾讯云

腾讯云为 OpenClaw 提供了专属活动页面，针对 QQ、企业微信、钉钉、飞书四大 IM 平台做了深度集成。

**部署方式**：轻量应用服务器 → 选择 OpenClaw 应用镜像 → 系统自动预装 OpenClaw 和 Docker 环境。

**配置流程**：
1. 购买轻量应用服务器，选择 OpenClaw（Clawdbot）应用镜像
2. 等待 1-2 分钟实例初始化
3. 浏览器访问服务器 IP，进入配置界面
4. 填入大模型 API Key（支持 OpenAI、Claude、GLM、DeepSeek 等）
5. 如需接入 QQ：在 QQ 开放平台创建机器人后，填入 AppID 和 AppSecret 即可绑定

**费用参考**：

| 配置 | 月费 | 适用场景 |
|------|------|---------|
| 2核2G | ~¥45-60 | 个人轻度使用 |
| 2核4G | ~¥100 | 日常使用推荐 |

- [腾讯云 OpenClaw 专属活动页](https://cloud.tencent.com/act/pro/lighthouse-moltbot)

## 阿里云

阿里云轻量应用服务器提供 OpenClaw 应用镜像，地域选择最丰富。

**部署方式**：轻量应用服务器 → 选择 OpenClaw 应用镜像 → 自动预装。

**优势**：
- 国内节点 + 中国香港、美国弗吉尼亚等海外节点，适合需要访问 Google、OpenAI 等海外服务的场景
- 新用户有入门优惠

**费用参考**：

| 配置 | 费用 | 适用场景 |
|------|------|---------|
| 入门（2核1G） | ~¥68/年 | 尝鲜体验 |
| 标准（2核4G） | ~¥50-100/月 | 日常使用推荐 |

> 建议选择 2G 以上内存配置以避免性能问题。

- [阿里云轻量应用服务器](https://www.aliyun.com/product/swas)

---

## 百度智能云

百度智能云提供 OpenClaw 极速简易版部署方案，与千帆平台深度整合。

**部署方式**：可视化配置面板，预装 OpenClaw 及常用 Skills。

**优势**：
- 一键接入文心大模型、DeepSeek、Qwen 等国产模型
- 不定期推出限时免费体验活动

**注意**：百度智能云的 OpenClaw 部署方案较新（2026 年 2 月推出），功能和文档可能还在完善中。

- [百度智能云](https://cloud.baidu.com/)

---

## 火山引擎

火山引擎（字节跳动旗下）提供 ECS 云服务器，需手动安装 OpenClaw。

**部署方式**：购买 ECS 实例 → SSH 登录 → 按[第七章](/cn/adopt/chapter7/)步骤手动安装。

**优势**：
- 与飞书等字节系产品协同使用体验最佳
- 「方舟 Coding Plan」提供 doubao-seed 等字节系模型 API
- 托管方案 [ArkClaw](https://www.volcengine.com/product/arkclaw) 提供完全免运维的云端 OpenClaw（详见[导言](/cn/adopt/#_6-1-三种使用方式)）

**注意**：无一键部署镜像，适合已有服务器运维经验的用户。

- [火山引擎](https://www.volcengine.com/)

## 自建 vs 托管怎么选？

| 维度 | 自建部署（本教程主线） | 云厂商一键部署 | ArkClaw 全托管 |
|------|---------------------|---------------|---------------|
| 控制权 | 完全掌控 | 大部分掌控 | 受限 |
| 运维成本 | 需自行维护 | 半托管 | 零运维 |
| 月费 | VPS 费用（¥30-100） | ¥45-100 | 按用量计费 |
| 灵活性 | 最高 | 中等 | 最低 |
| 上手难度 | 需看第七章 | 买了就能用 | 注册就能用 |
| 适合谁 | 想深入学习的用户 | 想省事但保留控制的用户 | 只想用不想管的用户 |

---

**最后提醒**：云厂商的服务内容和价格可能随时调整，部分"一键部署"服务可能有时间或名额限制。部署前建议访问官网确认最新信息，也可以先用本教程在本地跑通，再迁移到云端。
